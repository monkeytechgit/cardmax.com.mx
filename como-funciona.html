Preguntas frecuentes

Información antes de realizar tu pedido

Respuestas generales sobre personalización, archivos, producción y entrega.

¿Qué información debo enviar para comenzar?

Necesitamos conocer el producto, cantidad, logotipo, colores, datos que llevará cada tarjeta y cualquier referencia disponible.

¿Puedo solicitar cambios en la propuesta?

Sí. La propuesta se revisa antes de producción y se aplican los ajustes necesarios para aprobarla.

¿Cuándo comienza la producción?

La producción comienza después de confirmar el pedido, recibir la información necesaria y aprobar el diseño final.

¿Pueden organizar datos para varias personas?

Sí. Podemos trabajar con una lista de datos para personalizar múltiples tarjetas dentro del mismo pedido.

¿Cómo se coordina la entrega?

La entrega se coordina por paquetería o según las condiciones acordadas para el proyecto.
