# CardMax - Página Web

Página web profesional para CardMax, especialistas en impresión de credenciales y tarjetas PVC personalizadas.

## Estructura del Proyecto

```
pagina_web/
├── index.html                      # Página de inicio
├── servicios.html                  # Página de servicios
├── proceso.html                    # Página institucional - Proceso
├── acerca-de-nosotros.html        # Página institucional - Quiénes somos
├── envios.html                     # Página institucional - Envíos
├── credenciales-escolares.html     # Página de producto
├── gafetes-empresariales.html      # Página de producto
├── tarjetas-identificacion.html    # Página de producto
├── tarjetas-membresia.html         # Página de producto
├── tarjetas-regalo.html            # Página de producto
├── tarjetas-lealtad.html           # Página de producto
├── tarjetas-eventos.html           # Página de producto
├── tarjetas-presentacion.html      # Página de producto
├── tarjetas-rfid-qr.html          # Página de producto
├── tarjetas-personalizadas.html    # Página de producto
├── css/
│   ├── global.css                  # Estilos globales (navbar, botones, footer)
│   ├── inicio.css                  # Estilos específicos de inicio
│   ├── servicios.css               # Estilos específicos de servicios
│   ├── producto.css                # Estilos modulares para productos
│   ├── proceso.css                 # Estilos específicos de proceso
│   ├── acerca-de-nosotros.css     # Estilos específicos de nosotros
│   └── envios.css                  # Estilos específicos de envíos
├── js/
│   └── global.js                   # JavaScript global (navbar, FAQ, contadores)
└── README.md                      # Este archivo
```

1. Clona o descarga los archivos
2. Abre `index.html` en un navegador web
3. Para desarrollo, usa un servidor local (Live Server en VS Code)

## Notas de Desarrollo

- El proyecto usa CSS moderno con custom properties (variables CSS)
- JavaScript vanilla sin frameworks para máximo rendimiento
- Diseño mobile-first responsivo
- Optimizado para SEO y accesibilidad
- Preparado para integración con CMS o backend
